<p>Security Assessment and Testing has five objectives.</p>



<ol class="wp-block-list"><li>You should be able to implement security control in order to design and perform validation checks for security testing and assessment, to run smoothly the operations of organizational security requires you to devise various security testing strategies.</li><li>You should be able to devise correct approach for security testing including vulnerability assessments, log reviews and penetration testing where it requires.</li><li>This objective will help you to understand how to collect information regarding account management, key performance or risk indicator, backup and their verification and how to review these security management and operational controls. You will also be able to design and implement disaster recovery procedures.</li><li>Then you will be able to write security reports using automated and manual methods.</li><li>You should be able to perform security audits. These audits can be performed internally or can be conducted through specializes third parties.</li></ol>
